For the evaluation test set, the participant should submit a text file to the CodaLab platform which contains classification results for all utterances. And ScoreWWS or CER will be calculated and updated in the leaderboard. Each line of the file should be in the form as follows:

Task 1: <Utterance ID> <0 or 1>

Task 2: <Utterance ID> <Chinese characters sequence>

The official will provide utterance IDs. ‘0’ and ‘1’ in Task 1 represent the prediction results of the current sample.
